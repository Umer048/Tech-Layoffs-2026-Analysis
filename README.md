# 📊 2026 Tech Layoffs & AI Transformation
> **Goal:** Analyzing why tech giants are cutting jobs to fund AI-first strategies in Q1 2026.

## 🚀 Project Overview
This analysis investigates the workforce changes in the tech industry. The data reveals that companies are not just saving money—they are **investing in the future of AI.**

### 📈 Key Findings
* **61% AI Pivot:** Most layoffs are linked to AI Infrastructure shifts.
* **Mega Layoffs:** Analysis includes data from Oracle (30k) and Amazon (16k).
* **Market Sentiment:** Investors are rewarding companies that move toward AI efficiency.

### 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Matplotlib, Seaborn
* **Tool:** Jupyter Notebook
